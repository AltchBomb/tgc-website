Architecture:
global.css — shared design system (tokens, typography, buttons, FAQ, CTA, etc.) scoped to .tgc-page.
Each page links <link href="../global.css"> and its <style> block holds only page-specific rules.
partials/footer.html — shared footer loaded via fetch(). Run via npm start.

How to add a new page:
Save your HTML to pages/your-page-name.html, add class="tgc-page" to the wrapper div, link global.css, include the footer fetch snippet, then add a card to this hub.
